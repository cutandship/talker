# Talker 0.1.0 🎙️

**Говоришь — текст появляется там, где стоит курсор.** Голосовой ввод для Windows,
полностью офлайн: ни звук, ни слова не покидают твой компьютер. Русский — первым
классом ([GigaAM v3](https://github.com/salute-developers/GigaAM)), любой другой
язык — Whisper.

Первый публичный релиз.

### ⬇️ [Скачать TalkerSetup.exe (292 МБ)](https://github.com/cutandship/talker/releases/download/v0.1.0/TalkerSetup.exe)

## ⬇️ Что скачать

**`TalkerSetup.exe` (293 МБ)** — всё внутри: модель распознавания (GigaAM v3),
рантайм и зависимости. Работает офлайн сразу, ничего не докачивает. Python и что-либо
ещё ставить не нужно.

(Whisper для других языков — опционально, по кнопке «Скачать Whisper» в Настройках.)

## 🚀 Первый запуск

1. Скачай `TalkerSetup.exe` и запусти.
2. Пройди мастер (можно включить автозапуск и ярлык на рабочем столе).
3. Появится плавающая капсула и значок в трее — модель уже внутри, качать ничего не нужно.
4. **Зажми `Right Alt`, скажи что-нибудь, отпусти** — текст вставится туда, где курсор (Telegram, Word, браузер, IDE — куда угодно).
5. Без рук — `Ctrl+Alt+Space`.
6. Настройки и История — правый клик по капсуле или значку в трее.

## ⚠️ Если Windows ругается

Инсталлер пока без цифровой подписи, поэтому SmartScreen может показать «Система Windows защитила ваш компьютер». Нажми **«Подробнее» → «Выполнить в любом случае»**. (Подпись добавим в следующих релизах.)

## ✨ Возможности

- **Push-to-talk** (`Right Alt`) и **hands-free** (`Ctrl+Alt+Space` / wake-word «Hey Jarvis»).
- Умная пост-обработка **без ИИ-облака**: вырезает «ну»/«э-э», «двадцать пять процентов» → «25 %», голосовые команды форматирования (абзацы, списки, тире), словарь замен.
- Плавающая капсула с волной голоса и кнопками ✕/✓ — не крадёт фокус у активного окна.
- История с мгновенным поиском, экспорт в TXT/SRT/VTT/JSON.
- Транскрибация файлов и YouTube.
- Тихий режим (для шёпота), локальный HTTP API, лёгкие web-окна настроек.

## 🔒 Приватность

Распознавание, история и настройки — только на твоей машине. Нет телеметрии, облака и аккаунтов. Лог не содержит текста диктовок.

## 🖥️ Требования

Windows 10/11 (64-бит), микрофон.
